## Backboard.io

**Category:** AI Memory & RAG Platform
**Booth:** Main Hall - Booth #2

### About
Backboard.io is the #1 platform on LoCoMo (Long Context Memory) benchmarks. They provide unified APIs for AI memory, conversation threads, and document RAG.

### APIs Available
- **Memory:** Persistent user memory across sessions
- **Threads:** Conversation context management
- **Document RAG:** Upload and query documents
- **Models:** Integrated LLM inference

### Resources for Hackers
